Lip Sync Timing Generator

ElevenLabs → Rive

Enter your ElevenLabs API key, Voice ID, and text. The tool calls the ElevenLabs with-timestamps endpoint, maps each character to one of 10 English mouth shapes, and outputs a millisecond-accurate timing array plus ready-to-use JS code for your Rive state machine at runtime.

The 10 visemes cover the full range of English phonemes. Non-English text may produce less accurate results.

Remove your API key from ElevenLabs after each use.
